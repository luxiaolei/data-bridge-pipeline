# rithmic_raw dataset (EN/中文)

## EN
Raw snapshot dataset captured from the Rithmic orderflow API for XAUUSD/GC.

- Format: JSONL (one snapshot per line)
- Path pattern: `rithmic_raw/date=YYYY-MM-DD/orderflow_snapshot_HHMMSS.jsonl`
- Companion metadata file: `rithmic_raw/DATASET_DESCRIPTION.md`
- Primary usage: offline research, replay, feature extraction

## 中文
用于离线研究的 Rithmic 原始快照数据集（XAUUSD/GC）。

- 格式：JSONL（每行一个快照）
- 路径规范：`rithmic_raw/date=YYYY-MM-DD/orderflow_snapshot_HHMMSS.jsonl`
- 说明文件：`rithmic_raw/DATASET_DESCRIPTION.md`
- 主要用途：回放、特征构建、研究验证
